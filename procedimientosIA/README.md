# Procedimientos IA para Windsurf/Cascade

Esta carpeta contiene procedimientos automatizados para que la IA (Windsurf/Cascade) pueda ejecutar tareas complejas, repetitivas o monótonas de manera consistente.

## Propósito

- **Automatizar tareas repetitivas**: Reducir el trabajo manual en operaciones frecuentes
- **Estandarizar procesos complejos**: Garantizar consistencia en operaciones multifacéticas  
- **Optimizar flujo de desarrollo**: Permitir a la IA enfocarse en lógica mientras ejecuta procedimientos predefinidos

## Estructura

Cada procedimiento es un archivo .md con:
- Descripción clara del propósito
- Prerrequisitos necesarios
- Pasos detallados de ejecución
- Resultados esperados

## Uso

La IA puede referenciar estos procedimientos usando: "lee procedimientosIA/[nombre-procedimiento].md"

## Reglas

- Los procedimientos deben ser autocontenidos
- No deben requerir intervención humana una vez iniciados
- Deben manejar errores y casos límite
- Deben dejar logs o evidencia de su ejecución
