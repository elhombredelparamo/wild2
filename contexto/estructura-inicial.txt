FLUJO ACTUAL
------------
1. Al ejecutar el juego se carga scenes/main_menu.tscn (definido en run/main_scene)
2. Menú principal → CharacterSelectMenu → CharacterCreateMenu → NewGameMenu → WorldSelectMenu → GameWorld
3. Información del personaje mantenida consistentemente entre menús
4. Contexto claro sobre qué personaje está activo en cada momento

INPUT (project.godot):
- move_forward (W), move_back (S), move_left (A), move_right (D)
- jump (ESPACIO), crouch (CTRL), ui_cancel (ESC)

REQUISITOS PARA EJECUTAR
------------------------
- Godot 4.6.1 con soporte C#
- .NET SDK 8.0 instalado
- Abrir la carpeta del proyecto con Godot
- Ejecutar con F5 o el botón Play
